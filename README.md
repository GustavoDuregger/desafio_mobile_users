# desafio_mobile
App for list users

## Run
1. Clone the repository
```bash
https://github.com/GustavoDuregger/desafio_mobile_users.git
```
2. Enter the command in the project to download the dependencies
```bash
flutter pub get
```
3. Run main.dart
